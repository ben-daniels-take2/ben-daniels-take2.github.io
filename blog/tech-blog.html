what does it mean to display inline blocks ?



Display inline makes the element create inline-boxes which basically means that dimensions don’t apply and vertical margins no longer work. Vertical padding can be added but cannot increase the height of the element greater than it’s line height and so may overlap subsequent lines. Inline elements (or elements with a display of inline) can be wrapped around text (or other inline content) and used for styling but won’t affect the position of the text as such. inline-blocks on the other hand can have dimensions like block boxes but will shrink to fit the content rather than stretching full width as block boxes do. It allows for a series of block elements to behave as though they were on the same line (much like floating).
