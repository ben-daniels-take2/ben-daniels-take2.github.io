..Explain control flow and loops using an example process from everyday life..


The control flow is the order that a computer runs the code within a script. It flows down to the first line to the last line. unless it comes across conditionals or loops which change the control flow. Loops are used when you want something to be repeated, when you create a loop its important to include a code that tells the loop when to stop. Say you needed to hit something a certain amount of times to win a prize. a loop would cause you to keep repeating this, until something ended the loop. like when you win the prize and have hit the item a certain amount of times. you would program the loop to restart.
